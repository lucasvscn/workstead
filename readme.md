Workstead
=========

This is a modified version of [Laravel Homestead](http://laravel.com/docs/homestead?version=4.2).

Features included
=================

 - CentOS 6.5
 - Apache 2.2.15
 - PHP 5.5.18
 - MySQL 5.5.40
 - Composer
 - [PHP CS Fixer](http://cs.sensiolabs.org/)
 - phpMyAdmin
 - [Laravel Envoy](http://laravel.com/docs/4.2/ssh#envoy-task-runner)
 - Laravel Installer
 - Memcached
